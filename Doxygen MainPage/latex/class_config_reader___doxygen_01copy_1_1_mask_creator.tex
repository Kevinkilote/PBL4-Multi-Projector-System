{\footnotesize\ttfamily Config\+Reader\+\_\+\+Doxygen copy.\+Mask\+Creator.\+modify\+Intensity (\begin{DoxyParamCaption}\item[{}]{self,  }\item[{}]{image\+\_\+side }\end{DoxyParamCaption})}



@briefs This function modifies the intensity of the image stored in {\ttfamily self.\+result\+\_\+image} based on the {\ttfamily image\+\_\+side} parameter. 

The intensity modification is performed column by column, starting from the edge and moving towards the center of the image. The intensity of each pixel in the column is multiplied by an {\ttfamily intensity\+\_\+factor} which decreases linearly from 1 to 0 over the range of {\ttfamily self.\+\_\+\+\_\+mask}.

\begin{DoxyNote}{Note}
If {\ttfamily image\+\_\+side} is 1, the function modifies the intensity of the left side of the image. If {\ttfamily image\+\_\+side} is 0, the function modifies the intensity of the right side of the image. 
\end{DoxyNote}
